# Sales-volume-forecasting-and-analysis-system
#### 銷售量預測暨分析系統─以蘇州旅宿業為例
#### 發現蘇州的酒店有銷售量震盪的趨勢，因此和夥伴們一起開發了銷售量預測系統
＃ 系統介面：http://ec2-54-180-26-182.ap-northeast-2.compute.amazonaws.com/
